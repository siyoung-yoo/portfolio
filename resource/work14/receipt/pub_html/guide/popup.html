closeBtn 클래스 추가 시 팝업 닫힘
* 추가 기능이 필요 없는 단순 닫기버튼(.btnIconClose, 취소)에만 사용

fnPopupOpen( "btnName" , "popupName");
* btnName 없을 경우 클릭 이벤트 없이 팝업 오픈
　ex) fnPopupOpen( "" , "popupName");
　ex) fnPopupOpen( null , "popupName");

fnPopupClose( "popupName" , "focusEle");
* focusEle 있을 경우 해당 element 로 포커스 이동
